# Ejecutar en Desarrollo

1. Clone repo
2. Run 
```
npm install
```
3. Install Nest CLI
```
npm install -g @nestjs/cli
```
4. Run DataBase
```
docker-compose up -d
```

## Stack
* Mongo DB
* Nest
